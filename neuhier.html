Neu Hier?

Wie funktioniert die Website?

Auf der finden Sie eine allgemeine Übersicht über die verschiedenen Hilfsangebote im Raum Wandsbek. Durch das Klicken auf eine der Oberkategorien, gelangen Sie zu einer Unterkategorie. Sie können die einzelnen Angebote erreichen, indem Sie auf eine Unterkategorie klicken.

Durch klicken auf das Feld oben links gelangen Sie immer zur Startseite zurück. Unter dem Reiter Feedback können Sie den Veranstaltern eine Rückmeldung schicken, was Ihnen gefallen oder auch nicht gefallen hat. Außerdem können Sie veraltete Angebote auf diesem Wege an das Fachamt weiterleiten. Diese können dann gelöscht oder aktualisiert werden. Durch einen Klick auf Angebote gelangen Sie zu allen Angeboten.

Wie kann ich ein Feedback geben?

Indem Sie auf den Menüpunkt "Feedback" klicken, werden Sie zu unserer Feedback-Funktion weitergeleitet. Dort können Sie in dem oberen Fenster einen Betreff angeben und in dem unteren Ihr Feedback schreiben. Durch das Betätigen des Abschicken-Buttons wird ihr Feedback automatisch an eine festgelegte E-Mail-Adresse gesendet.

Warum finde ich bei der Suche nichts?

Das kann natürlich mehrere Gründe haben. Zum einen gibt es keinen Ihrer Suche entsprechenden Eintrag, oder Sie haben sich vertippt. Also am besten die Rechtschreibung nochmals überprüfen.
