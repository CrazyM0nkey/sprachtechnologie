Anleitung für den Admin Bereich?

Was für Funktionen stehen für mich als Admin bereit?

Als Admin können Sie verschiedene Operationen durchführen.

Dazu gehören:
- Erstellen von neuen Einträgen
- Erstellen von neuen Kategorien inkl. Bildupload
- Löschen von Einträgen
- Löschen von Kategorien
- Feedback der Nutzer einsehen.

Wie erstelle ich einen Eintrag?

Wenn Sie sich als Admin eingeloggt haben, landen Sie auf der Hauptseite des Admin-Bereichs. Hier haben Sie die Möglichkeit unter der Überschrift "Eintrag erstellen" einen neuen Eintrag in die Datenbank vorzunehmen. Dazu füllen Sie einfach die verschiedenen Felder mit Ihren Daten aus. Dabei sollten Sie darauf achten immer einen Träger, eine Straße, eine Hausnummer, eine Postleitzahl, einen Ansprechpartner mit mindestens einer Möglichkeit diesen zu erreichen, sowie eine Beschreibung des Angebots anzugeben.

Die Beschreibung sollte die Sprachen beeinhalten, in der die Veranstaltung abgehalten wird und darüber hinaus auch die Zielgruppe enthalten und nicht länger als 700 Zeichen sein. Achten Sie dabei wenn möglich einfache Sprache zu verwenden, da die Suchfunktion dieses Feld nach Schlagwörtern durchsucht. Außerdem ist es notwendig eine Unterkategorie anzugeben, in der der Eintrag erstellt werden soll.

Durch Betätigung der Schaltfläche "Eintragen" wird die Erstellung des Angebots abgeschlossen. Sollte es keine passende Kategorie für Ihren Eintrag geben, so ist es notwendig zunächst eine neue Kategorie anzulegen.

Wie erstelle ich eine neue Kategorie?

Eine neue Kategorie können Sie unter der Überschrift "Kategorieverwaltung" erstellen. Wir unterscheiden zwischen Ober- und Unterkategorie. Die Oberkategorien werden auf der Startseite angezeigt. Die Unterkategorien erreicht man durch das Anklicken einer Oberkategorie. In einer Unterkategorie werden die verschiedenen Einträge angezeigt.

Wenn Sie eine Oberkategorie erstellen möchten, müssen sie unter "Unterkategorie wählen" die Option Sozialraum anklicken.
Um ein Bild für eine Kategorie festzulegen, klicken sie unter "Bild Upload" auf "Datei auswählen" und suchen in Ihrem Dateisystem das zugehörige Bild aus. Anschließend klicken Sie auf Hinzufügen, um das Erstellen der Oberkategorie abzuschließen.
Das Erstellen einer Unterkategorie funktioniert analog dazu, nur dass man anstelle des Sozialraums die zugehörige Oberkategorie auswählen muss. Dieses Vorgehen ist notwendig, da wir auf diese Weise die Datenbank strukturiert haben.

Wie lösche ich einen Eintrag?

Unter der Überschrift "Einträge löschen" werden zwei Suchmöglichkeiten angeboten.
Auf der linken Seite ist ein Dropdown-Menü mit dessen Hilfe die Eiträge gefiltert werden können. Dazu wählen Sie eine Kategorie und drücken auf den Button Anzeigen.
Auf der rechten Seite gibt es ein Suchfeld, in dem Sie nach einem Träger suchen können. Dazu tippen Sie in das Suchfeld den Namen des Trägers und drücken ebenfalls auf den Button Anzeigen. Halten sie die Suchanfrage möglichst einfach.

Nun sollten Sie eine Liste von Einträgen sehen können. Aus dieser können sie beliebig viele Angebote auswählen und löschen.

Wie lösche ich eine Kategorie?

Sie wählen einfach aus dem Dropdown-Menü die zu löschende Kategorie aus und drücken auf die Schaltfläche "Löschen".

Dabei ist zu beachten, dass die Kategorie "Sozialraum" niemals gelöscht werden darf. Außerdem gilt, dass wenn eine Kategorie gelöscht wird, alle Einträge und alle zugehörigen Unterkategorien gelöscht werden. Also stellen Sie stets sicher, dass Sie wirklich alle Einträge einer Kategorie löschen wollen.

Wie kann ich mir die Feedback Nachrichten ansehen?

Durch die Betätigung des "Feedback auslesen"-Buttons in der Menüleiste im Admin-Bereich gelangen Sie zu der "Feedback Ausgabe". Dort können sie für einen von Ihnen bestimmten Zeitraum, die Feedbacks anzeigen lassen. Dazu wählen sie den Monat und das Jahr in dem jeweiligen Dropdown-Menü aus und klicken auf "Anzeigen".
Nun sehen Sie die Nachrichten für den angegebenen Zeitraum.
